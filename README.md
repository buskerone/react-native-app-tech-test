# Belin App
This is a demo app for a technical test.

## Instructions

 1. `yarn` or `yarn install`
 2. Install pods `cd ios` -> `pod install`
 3. iOS: `react-native run-ios`
 4. Android: `react-native run-android`

## Folder structure

 - src
	 - assets
	 - components
	 - navigation
	 - redux
		 - actions
		 - reducers
	- screens
		- RegistrationScreens
			- RegistrationFormScreen
			- RegistrationLookScreen
